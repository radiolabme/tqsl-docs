> Dedicated to keeping the technology for digitally signed eQSLs freely available to the amateur radio community

## Mission

The mission of the Trusted QSL Group is to keep eQSL an open and free activity for amateur radio operators by promoting the **Digital Signature Standard (DSS)**.

The Digital Signature Standard (DSS) specifies algorithms appropriate for applications requiring a digital, rather than written, signature. A digital signature is represented in a computer as a string of binary digits. A digital signature is computed using a set of rules and a set of parameters such that the identity of the signatory and integrity of the data can be verified. An algorithm provides the capability to generate and verify signatures.

## Goal

The goal of the Trusted QSL Group is to set the standards and create the necessary tools and software needed to implement the Trusted DSS model.
